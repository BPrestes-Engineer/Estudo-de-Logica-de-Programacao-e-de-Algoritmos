Estudos de Algoritmos e Lógica de Programação

Este repositório documenta minha jornada de aprendizado em **Algoritmos e Lógica de Programação**, iniciada com o uso do **VisualG (Portugol)** e baseada no curso de **210 horas** - "Lógica Programação VisualG, C++, Python, C#, Java + Projetos " - do professor Clevison Santos na Udemy.  

📚 Conteúdo do Curso
O curso cobre uma ampla gama de tópicos e linguagens, incluindo:

- **VisualG (Portugol)**: fundamentos da lógica de programação  
- **C++**: estruturas básicas, funções e Programação Orientada a Objetos  
- **Python**: listas, dicionários, funções, classes e automação (RPA)  
- **Java**: estruturas de controle, arrays, matrizes e interface gráfica  
- **C#**: fundamentos, POO e integração com banco de dados  
- **SQLite**: integração com projetos reais  
- **Android Studio (Java)**: desenvolvimento de jogos simples  

🎯 Objetivos do Repositório
- Consolidar fundamentos de lógica de programação.  
- Praticar a tradução de raciocínio lógico em código.  
- Documentar minha evolução em diferentes linguagens.  
- Criar uma base sólida para projetos aplicados e oportunidades profissionais.

🚀 Próximos Passos
- Expandir os exercícios para outras linguagens (C++, Python, Java, C#).  
- Criar pequenos projetos aplicados com interface gráfica e banco de dados.  
- Documentar cada projeto com README próprio, explicando objetivos e funcionamento.  

📌 Observação
Este repositório é parte da minha jornada de aprendizado.  
Ele não representa projetos finais, mas sim exercícios e práticas que mostram minha evolução passo a passo.
E lembre-se, o importante é nunca parar de se movimentar para se manter equilibrado ❤
